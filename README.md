# Streaming Services — Data Analysis

Проект по анализу данных о стриминговых сервисах: разведочный анализ, обработка данных и визуализация.

## О проекте

На данных о стриминговых платформах проводится анализ и визуализация — от очистки и подготовки данных до наглядных интерактивных графиков и выводов.

## Что внутри

- **`pyprojectSimonova.ipynb`** — основной ноутбук проекта (анализ и визуализация)

## Стек

Python, pandas, NumPy, matplotlib, seaborn, plotly.

## Как запустить

```bash
pip install pandas numpy matplotlib seaborn plotly
jupyter notebook pyprojectSimonova.ipynb
```
